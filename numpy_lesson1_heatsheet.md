## üîπ –ê—Ä–∏—Ñ–º–µ—Ç–∏–∫–∞ –∏ Broadcasting

```python
a = np.array([1, 2, 3])
```

## üîπ –ú–∞—Ç—Ä–∏—Ü—ã

```python
A = np.random.randint(1, 10, (3, 3))
B = np.random.randint(1, 10, (3, 3))

A + B           # –ø–æ—ç–ª–µ–º–µ–Ω—Ç–Ω–æ–µ —Å–ª–æ–∂–µ–Ω–∏–µ

```

## üîπ –ê–Ω–∞–ª–∏–∑ —Å–ª—É—á–∞–π–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã

```python
M = np.random.rand(10, 5)   # 10 —Å—Ç—Ä–æ–∫ √ó 5 –∫–æ–ª–æ–Ω–æ–∫

M.mean(axis=0)   # —Å—Ä–µ–¥–Ω–µ–µ –ø–æ –∫–æ–ª–æ–Ω–∫–∞–º
M.max(axis=1)    # –º–∞–∫—Å–∏–º—É–º –ø–æ —Å—Ç—Ä–æ–∫–∞–º
M.min(axis=1)    # –º–∏–Ω–∏–º—É–º –ø–æ —Å—Ç—Ä–æ–∫–∞–º
```

### –ù–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏—è
```python
M_norm = (M - M.min(axis=0)) / (M.max(axis=0) - M.min(axis=0))
```

---

## üîπ –í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è

```python
plt.imshow(M_norm, cmap="viridis", aspect="auto")
plt.colorbar(label="–ó–Ω–∞—á–µ–Ω–∏–µ")
plt.title("Heatmap –Ω–æ—Ä–º–∞–ª–∏–∑–æ–≤–∞–Ω–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã")
plt.show()
```
